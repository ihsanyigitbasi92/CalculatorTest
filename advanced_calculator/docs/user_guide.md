# User Guide

## Getting Started
- Launch the application using `python app/main.py`.
- Use the GUI to perform calculations.

## Features
- **Basic Arithmetic**: Use the calculator interface for addition, subtraction, multiplication, and division.
- **Scientific Functions**: Access trigonometric and logarithmic functions from the scientific tab.
- **Statistical Operations**: Calculate mean, median, mode, and standard deviation using the statistics tab.
- **Graphical Representation**: Plot graphs by entering functions in the plot tab.
- **Unit Conversion**: Convert units using the conversion tab.
- **History**: View and manage previous calculations in the history tab.
- **Custom Functions**: Define and use custom functions in the custom functions tab.

## Error Handling
- The application provides error messages for invalid inputs.
