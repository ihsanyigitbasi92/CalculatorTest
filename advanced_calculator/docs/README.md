# Advanced Calculator

## Overview
The Advanced Calculator is a Python-based application that provides a wide range of mathematical operations including basic arithmetic, scientific functions, statistical operations, graphical representation, unit conversion, and custom functions. It features a user-friendly GUI for ease of use.

## Installation
To install the required dependencies, run:
```bash
pip install -r requirements.txt
```

## Usage
To start the application, run:
```bash
python app/main.py
```

## Features
- Basic Arithmetic Operations: Addition, subtraction, multiplication, and division.
- Scientific Functions: Trigonometric functions, logarithmic functions, exponential functions.
- Statistical Operations: Mean, median, mode, standard deviation.
- Graphical Representation: Plotting graphs for functions.
- Unit Conversion: Convert between different units of measurement.
- History and Memory: Store previous calculations and recall them.
- Custom Functions: Define and save custom functions.

## Extensibility
The application supports a plugin system for adding new features.
